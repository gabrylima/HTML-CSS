Negrito / Itálico

negrito

Para chamar a atenção do leitor para uma determinada palavra de importância no texto, usamos 'Negrito'

Itálico

Para por em itálico, usamos <EM> para dar ênfase

Marca texto

Vamos usar a tag <MARK> para deixar a palavra em dastaque marcada,Meu nome é Gabriela!

E para trocar de cor o marcador só é por dentro do Mark a palavra style, Meu nome é gabriela!

Texto grande/ pequeno

Outras tags muito usadas são as que deixam um texto grande e um texto pequeno, o grande se usa big e a o pequeno small.

Meu nome é Gabriela! e Meu nome é Gabriela!

Texto excluido

Vai-se usar o <del> para excluir, Texto excluido
